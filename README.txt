Projet : Exploitation de Nouvelles Bases de Données d'Assurance
Ce projet vise à analyser et exploiter des bases de données relatives aux sinistres et contrats d'assurance automobile. L'ensemble de l'analyse est réalisé en utilisant le langage de programmation R, offrant ainsi une gamme complète d'outils pour l'analyse, la visualisation, la tarification et la prédiction.

Outils Utilisés
R : Utilisé pour l'ensemble de l'analyse statistique et des opérations de traitement des données.
R-Shiny : Employé pour créer une interface utilisateur permettant d'interagir avec les résultats de l'analyse.
Plateforme Utilisée
Ce projet a été développé en utilisant l'environnement R Studio, offrant une suite complète d'outils pour la manipulation et l'analyse de données.

Compétences Mises en Œuvre
Ce projet a permis de mettre en œuvre plusieurs compétences clés en assurance et en analyse statistique :

Statistiques Descriptives : Analyse descriptive des données pour comprendre les tendances et les caractéristiques clés.
Modélisation par GLM (Generalized Linear Models) : Utilisation de modèles statistiques pour analyser les relations entre les différentes variables.
Tarification et Proposition d'une Prime : Évaluation des risques et détermination de primes appropriées.
Création d'une Interface avec R-Shiny : Développement d'une interface interactive pour explorer les résultats de l'analyse.
Bases de Données Utilisées
Trois bases de données ont été collectées et exploitées dans le cadre de cette étude :

Base Prem : Cette base contient 30 variables et 52 372 lignes, correspondant à des assurés. Les données couvrent les années 2003 et 2004, fournissant des informations détaillées sur les assurés.
Base Sev : Cette base contient 8 931 observations, correspondant à des sinistres et comprend 6 variables. Ces données fournissent des informations sur les sinistres des assurés.
Base Freq : Bien que cette base contienne des informations complémentaires, elle n'a pas été utilisée dans le cadre de cette étude.
L'objectif de cette analyse était d'explorer, modéliser et comprendre les interactions entre les sinistres et les contrats d'assurance automobile, ainsi que de fournir des recommandations basées sur les résultats obtenus.
